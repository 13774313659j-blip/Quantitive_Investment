# A股多因子选股模型 (Multi-Factor Stock Selection Model)

> 基于 Python 的多因子量化选股系统，融合价值/动量/质量/规模等多类因子，使用机器学习方法（逻辑回归、随机森林）合成综合得分，并通过滚动训练+回测验证策略有效性。

## 项目简介

本项目实现了一个完整的 A 股多因子选股 Pipeline，覆盖从数据生成、因子构建、预处理、有效性检验、机器学习合成、到组合回测与业绩评价的全流程。模型在合成数据上验证有效，随机森林策略相对基准取得显著超额收益。

## 核心流程

```
数据生成 → 因子构建 → 因子预处理 → 因子检验 → ML合成 → 组合回测
   1         2           3            4          5         6
```

1. **数据生成**：用几何布朗运动(GBM)合成 200 只股票 2020–2023 的日线行情和基本面数据，注入小市值/价值/动量/质量等因子效应。
2. **因子构建**：构建 9 类因子 — EP、BP（价值）、SIZE（规模）、MOM20/60（动量）、REV20（反转）、VOL20（波动率）、TURN20（流动性）、ROE（质量）。
3. **因子预处理**：MAD 去极值 → Z-score 标准化 → 行业/市值中性化。
4. **因子检验**：计算 IC（信息系数）、ICIR、分层回测多空收益。
5. **ML 合成**：Walk-Forward 滚动训练逻辑回归与随机森林，对比等权合成。
6. **组合回测**：每月调仓选 Top-20，等权配置，计算年化收益/夏普/回撤。

## 关键技术点

- **Walk-Forward 滚动训练**：用过去12个月数据训练，预测下1个月，避免数据泄露
- **二分类标签**：以"未来收益是否高于截面中位数"为标签，避免回归对极端值敏感
- **因子中性化**：剥离行业和市值效应，提取"纯因子"
- **MAD去极值**：相比3σ法更稳健，不受极端值影响

## 关于数据

本项目出于"可复现、无需联网"的考虑，使用合成数据。数据生成过程中刻意注入了真实市场中常见的因子效应（小市值溢价、价值溢价、动量效应、质量溢价），使模型能够演示出有效的因子选股逻辑。

实际生产环境中，可将 `data_generator.py` 替换为 tushare / baostock / akshare 等真实数据源，接口签名保持一致即可。
